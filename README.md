# Mineland Forsaken

## English
A 3D asymmetric multiplayer horror game built with Rust and Bevy engine.

### Features
- Main menu with play, settings and quit options
- Lobby system for player matchmaking
- Basic player character with movement controls
- Simple enemy AI
- Multiplayer networking (in development)

### Setup
1. Install Rust from [rust-lang.org](https://www.rust-lang.org/)
2. Clone this repository
3. Run `cargo run` to start the game
